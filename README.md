## 🔍 분석 방법

### 1. 데이터 시각화 (`통계분석_graph.ipynb`)
- 일별 확진자 추이 시계열 그래프 (기간별 분할 시각화)
- 월별 신규 확진자 / 누적 확진자 추이
- 월별 온라인 음식 서비스 거래액 추이 (총거래액 / 종합몰 / 전문몰)

### 2. 상관관계 시각화 및 분석 (`graph_merge.ipynb`)
- 월별 데이터 병합 (`pd.merge`)
- **Scatter plot (relplot)**: 신규 확진자 수 — 총거래액 산포도
  - 확진자 수가 일정 임계값 초과 시 선형 경향 이탈 구간 별도 분리
- **lmplot**: 회귀선 시각화 (신규 확진자 / 누적 확진자 각각)
- **Jointplot**: 회귀선 + 밀도 분포 동시 시각화

### 3. 통계 분석
- **피어슨 상관계수** 산출
  - `infected` — `total` : **r ≈ 0.895**
  - `accrue` — `total` : **r ≈ 0.870**
- **OLS 선형 회귀 모델** (`statsmodels`)
  - `lm_1`: `total ~ infected`
  - `lm_2`: `total ~ accrue`
- **AIC 기반 모델 선택** → `lm_1` (신규 확진자 모델) 채택
- **모델 평가**
  - 결정계수 R² = **0.8027** (설명력 약 80.27%)
  - Prob(F-statistic) < 0.05 → 통계적으로 유효한 회귀 모델
- **잔차 분석** (정규성 검정, 자기상관 검정)
  - Prob(Omnibus), Prob(JB) 정규성 만족
  - Durbin-Watson = 1.724 → 자기상관 없음

## 📈 주요 결과

- 코로나19 **신규 확진자 수**와 온라인 음식 배달 **총거래액** 사이에는 **양의 상관관계** 존재 (~2021년 8월)
- 신규 확진자 수가 일정 수준(임계값)을 초과하면 거래액이 선형 경향에서 이탈하는 현상 관측 (2021.8~)

| 2021.8~ | 경향 이탈 | 배달 플랫폼 수수료 인상 → 배달 수요 감소 |
